= Growatt PV inverter

Implementation of the Growatt PV inverters.

Tested on
- Growatt NIC

With versions:
- UI Version: 01.18.05255
- MC version: 01.47
- IOC version: 01.45

Older versions had problems with the implementation of sunspec.

Implemented Natures:
- ElectricityMeter
- ManagedSymmetricPvInverter
